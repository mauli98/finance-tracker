Finance Tracker
Overview
Finance Tracker is a comprehensive application designed to help users manage their personal finances efficiently. It provides tools for tracking income, expenses, and generating financial reports.

Features
Income and Expense Tracking: Easily log and categorize your income and expenses.
Budget Management: Set budgets for different categories and track your spending against them.
Financial Reports: Generate detailed reports to understand your spending habits and financial health.
Notifications: Receive alerts for upcoming bills, budget limits, and other important financial events.
User-Friendly Interface: Intuitive design for seamless user experience.
Installation
Prerequisites
Node.js
PHP
Composer
MySQL
Steps
Clone the Repository:

Copy
git clone https://github.com/mauli98/finance-tracker.git
Navigate to the Project Directory:

Copy
cd finance-tracker
Install Dependencies:

Copy
composer install
npm install
Set Up the Database:

Create a new MySQL database.
Update the .env file with your database credentials.
Run the migrations:
Copy
php artisan migrate
Start the Application:

Copy
php artisan serve
npm run dev
Usage
Register/Login: Create an account or log in to your existing account.
Dashboard: View an overview of your financial status.
Transactions: Add, edit, or delete income and expense transactions.
Budgets: Set up and manage your budgets.
Reports: Generate and view financial reports.
